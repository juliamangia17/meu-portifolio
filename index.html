Meu Catálogo virtual: Os melhores filmes

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção pessoal de filmes favoritos. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus filmes preferidos, fornecendo uma descrição breve de cada um, incluindo o diretor, o ano de publicação e uma opção de assistir.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de filmes em uma seção flexível (.filmes). Cada livro é destacado em seu próprio contêiner (.filme), mostrando uma imagem da capa, detalhes do filme e onde assistir. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de livros.

Meu Catálogo virtual: Os melhores filmes

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simula uma experiência de idrarias de uma laboratório. O objetivo é ajudar os jogadores em um cenário fictício que aborde quais vidrarias devem ser usadas.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, caixa de perguntas, alternativas de resposta, e o resultado final baseado nas escolhas do jogador. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando o DOM para atualizar o conteúdo da página conforme as escolhas do jogador, e determinando o fluxo da narrativa através de uma série de perguntas e respostas.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de filmes em uma seção flexível (.filmes). Cada filme é destacado em seu próprio contêiner (.filme), mostrando uma imagem da capa, detalhes do filme e um link para assistir. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Os scripts JavaScript incluem módulos para gerar nomes aleatórios, definir as perguntas e as lógicas de jogo, como iniciar o jogo, apresentar perguntas e alternativas, processar as escolhas dos jogadores, e exibir o resultado final. Este design modular facilita a manutenção e a expansão futura do jogo. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa que não apenas entretém, mas também educa os jogadores sobre as vidrarias de laboratório.

Meu Catálogo virtual: Os melhores filmes

O projeto é uma aplicação educativa interativa focada em mostrar cenas de crimes que ocorreram. Utiliza uma série de scripts associados a diferentes atores (sprites) e cenários para criar uma experiência de aprendizado envolvente. O objetivo é familiarizar os usuários com tópicos como as fases da Lua, eclipses, a visibilidade de diferentes constelações dependendo da localização geográfica na Terra, e a forma da Terra, através de interações e visualizações dinâmicas.

Nesse cenário contem amostras onde seram coletadas por peritos, para serem analisados em laboratórios, para descobrir o responsavel por esse ato.

A importância das ferramentas utilizadas, como os blocos de código visual do Scratch, reside na sua acessibilidade e na capacidade de engajar aprendizes de diferentes idades no pensamento computacional e na exploração de conceitos científicos de forma interativa. O uso de cenários, movimentos, e diálogos dinâmicos ajuda a criar uma experiência de aprendizado mais rica e memorável, encorajando os usuários a explorar e interagir com o conteúdo de maneira ativa, em vez de passivamente.
